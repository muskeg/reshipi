# クレーム・ブリュレ

<div class="recipe-hero" markdown>
<div class="recipe-intro" markdown>

滑らかなカスタードクリームの上に、パリッとしたカラメルをのせたフランスの定番デザート。

| | |
|---|---|
| **人数** | 4人分 |
| **準備時間** | 20分 |
| **調理時間** | 45分 |

## 材料

- 卵黄 4個
- 砂糖 <span class="qty" data-qty="80" data-unit="g" data-type="sugar">80 g</span>（トッピング用に追加分も用意）
- 生クリーム <span class="qty" data-qty="480" data-unit="ml" data-type="cream">480 ml</span>
- バニラエッセンス <span class="qty" data-qty="1" data-unit="tsp">小さじ1</span>

## 作り方

1. オーブンを160°Cに予熱する。
2. 卵黄と砂糖を滑らかになるまで泡立てる。
3. 生クリームを沸騰直前まで温め、卵液に少しずつ加えながら混ぜる。
4. バニラエッセンスを加える。
5. ラメキンに注ぎ、天板に並べる。天板にラメキンの半分の高さまでお湯を注ぐ。
6. 40〜45分、中心がわずかに揺れる程度まで焼く。
7. 冷蔵庫で最低2時間冷やす。
8. 砂糖を振りかけ、バーナーまたはグリルでキャラメリゼしてから召し上がれ。
