# クレーム・ブリュレ

| | |
|---|---|
| **人数** | 4人分 |
| **準備時間** | 20分 |
| **調理時間** | 45分 |

## 材料

- 卵黄 4個
- 砂糖 大さじ3（トッピング用に追加分も用意）
- 生クリーム 480ml
- バニラエッセンス 小さじ1

## 作り方

1. オーブンを160°Cに予熱する。
2. 卵黄と砂糖を滑らかになるまで泡立てる。
3. 生クリームを沸騰直前まで温め、卵液に少しずつ加えながら混ぜる。
4. バニラエッセンスを加える。
5. ラメキンに注ぎ、天板に並べる。天板にラメキンの半分の高さまでお湯を注ぐ。
6. 40〜45分、中心がわずかに揺れる程度まで焼く。
7. 冷蔵庫で最低2時間冷やす。
8. 砂糖を振りかけ、バーナーまたはグリルでキャラメリゼしてから召し上がれ。
